h2. About

"Immutable Collections Library for Javascript" is powerful set of immutable collection classes that will make it more natural to write javascript programs in a functional manner using features like higher-order functions.

The library is written in Coffeescript[1], a small language that compiles to JavaScript, as it adds a few semantics simplifies the code.

This is a product of the bachelor project of Mads Hartmann Jensen under supervision of Peter Sestoft.
