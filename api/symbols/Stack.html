Class Detail
Stack(elements)
Stack provides the implementation of the abstract data type Stack based on a List. The Stack contains the following operations: 
  --------------------------------------------------------
  Core operations of the Stack ADT
  --------------------------------------------------------
  pop()                                               O(1)
  push( elem )                                        O(1)
  top()                                               O(1)
  --------------------------------------------------------
Parameters:
elements
A repeatable argument with the elements you want on the Stack. The last element will be on top of the stack.
